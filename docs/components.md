# UI Components

Inventory of reusable UI components. Add new components as they are built.

## Format

```
### ComponentName
**Location:** `app/views/components/` or `app/helpers/`
**Usage:**
  Brief description of what it renders and when to use it.
**Example:**
  Code snippet showing how to use it in a view.
**Variants:**
  List of supported variants/options.
```
